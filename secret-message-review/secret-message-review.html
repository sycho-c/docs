비밀 메시지 기능 리뷰

내부 사용자가 특정 외부 파트너에게만 보이는 비밀 메시지 — PC · 모바일 · 파트너 3개 화면

1기능 개요

협업 대화방에는 내부 사용자(상담사)와 외부 파트너가 함께 참여한다. 비밀 메시지는 내부 사용자가 특정 외부 파트너에게만 보이도록 보내는 메시지다. 권한이 없는 참여자에게는 내용이 노출되지 않는다.

  • 열람 정책 — 내부 사용자는 비밀 메시지를 항상 열람한다. 외부 파트너는 지정된 수신자만 열람한다.
  • 권한 없는 참여자 — 본문 대신 잠금 안내 버블(자물쇠 아이콘 + “비밀 메시지로 작성된 내용입니다”)만 보인다. ※ 제품의 “*** 치환” 기능과는 무관한, 별도의 열람 제한 표시다.
  • 작성 방법 — 입력창에 ! 를 입력하면 비밀 메시지를 받을 파트너 선택 화면이 뜬다. (파트너 화면은 + 도구 메뉴로도 진입)

2동작 규칙

2.1 누가 무엇을 보는가 — 열람 권한

같은 비밀 메시지라도 보는 사람에 따라 노출이 달라진다.

보는 사람내부 사용자에게 보낸 비밀본인이 받는 비밀다른 파트너에게 보낸 비밀
내부 사용자열람열람열람
외부 파트너 — 수신자—열람🔒 잠금 안내
외부 파트너 — 비수신자🔒 잠금 안내—🔒 잠금 안내
핵심 원칙 — “내부 사용자는 항상 열람, 외부 파트너는 지정된 수신자만 열람.” 권한이 없으면 본문·첨부가 노출되지 않고 잠금 안내 버블로만 표시된다.

2.2 실시간 수신 동작

  • 비밀 메시지가 실시간으로 도착하면, 열람 권한이 있는 당사자에게는 잠깐의 가림 없이 본문이 바로 보이도록 처리했다.
  • 보낸 직후/받는 순간에도 일반 메시지와 비밀 메시지의 모습이 뒤바뀌어 깜빡이지 않도록 안정화했다(버블 색·답장 표시 포함).

4.1 화면 요소

요소표시 내용비고
비밀 메시지 버블 네이비 #373A73 배경 + 흰 본문일반 버블과 즉시 구분
비밀 멘션 텍스트 라벤더 #9DA0D9네이비 배경 위 가독 확보
잠금 안내 버블자물쇠 아이콘 + “비밀 메시지로 작성된 내용입니다”권한 없는 참여자 전용 표시
작성 안내 배너입력폼 위 전체너비 “OO에게 비밀 메시지 전송” + 닫기(X)답장 안내와 단일 영역으로 통합(겹침 방지)
대화 목록 / 알림마지막 메시지가 비밀이면 “비밀 메시지”로 표기파일이 “파일”로 보이는 방식과 동일

4.2 작성 흐름

  • 내부 사용자(PC·모바일) — 입력창에 ! → 받을 파트너 선택 → 상단에 ‘OO에게 비밀 메시지 전송’ 안내 표시 → 전송.
  • 파트너 — ! 입력 또는 + 도구 메뉴의 ‘비밀 메시지’로 진입(기존 🔒 토글은 제거).
  • 비밀 답장 — 비밀 메시지에 답장하면 일반 답장과 구분되는 ‘비밀 답장’ 문구로 표시.

4.3 알림·멘션 정합성 (눈에 보이지 않는 동작)

비밀 메시지를 받을 수 없는 파트너에게는 멘션 알림과 안 읽은 멘션 뱃지(@N)가 생기지 않도록 집계를 바로잡았다. (이전에는 @전체 같은 단체 멘션이 펼쳐질 때 권한 없는 파트너에게도 뱃지가 잘못 붙었다.)

  • 내부 사용자는 항상 받을 수 있으므로 멘션 알림이 그대로 유지된다.
  • 외부 파트너는 비밀 메시지 수신자에게만 멘션·뱃지가 생긴다.
